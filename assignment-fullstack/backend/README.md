# Backend Assignment

A Scalable REST API developed with Node.js, Express, and MongoDB.

## Features
- User registration and login with JWT authentication
- Role-based access control (user vs admin)
- CRUD operations for a secondary entity (Tasks)
- centralized error handling and API validation
- Swagger API documentation

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Create a `.env` file in the root directory
4. Start the server (Dev): `npm run dev`
5. Start the server (Prod): `npm start`
